# Life OS Product Brain v9.7

## Phase Update

Life OS v9.7 closes two important product gaps from the current roadmap:

- iPhone and iPad install guidance
- stronger dashboard entry points and weekly context

## New In This Version

- Apple-device install help modal for manual Add to Home Screen flow
- Weekly summary card with spend, top category, and completed habits
- One-tap dashboard quick actions for Expense, Grocery, Borrowed, and Note
- Version bump across app shell, service worker, manifest, tools, and translations

## Why This Matters

This version improves discoverability and daily usage without changing the architecture. It makes the app easier to install on Apple devices and easier to use as a daily command-driven workspace.

## Recommended Next Step

For v9.8, the strongest next move is to add the two best engagement tools from the roadmap:

- Tasks / Today List
- Daily Journal
